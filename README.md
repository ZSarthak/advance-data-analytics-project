# advance-data-analytics-project
In this project, we perform different types of advance analysis on the database to understand the metrics change, growth or loss over time and create data segments to help build business reports.
We perform the following analysis:
  - Change Over Time Analysis
  - Cumulative Analysis
  - Performance Analysis
  - Part to Whole Analsys
  - Data Segmentation

In the end, we create Views using this analysis that can be used to create business reports and be extracted to reporting tools like Excel, PowerBi and Tableau.
